# AGENTS.md

## Project Overview

**sodagreen.me** — 苏打绿乐团（Sodagreen）粉丝主页。作为根域名 `sodagreen.me` 的落地页，这是一个亮点展示页面，面向所有设备（桌面、平板、手机）做响应式适配。

## Core Requirements

### 1. 随机歌词
- 页面每次加载时随机展示一句苏打绿的歌词
- 歌词需准备一个数据集（数组），包含经典歌词

### 2. 全屏背景图
- 一张大幅背景图作为页面视觉主体
- 需考虑加载性能（压缩、懒加载或渐进式加载）
- 移动端可能需要使用裁切或不同比例的版本

### 3. 人物 LOGO
- 用户将提供的苏打绿成员趣味 LOGO 图片
- 需预留图片资源位，展示方式待定（可能为横向排列或悬浮动画）

### 4. 项目卡片（重点）
- 展示用户做过的与苏打绿相关的项目
- 每张卡片包含：项目标题、简述、链接
- 卡片需有视觉吸引力，支持 hover 交互效果
- 响应式布局：桌面多列、移动端单列

### 5. 外部链接
- 链接到苏打绿乐团官方网站
- 链接到乐团社交媒体（微博、Instagram、Facebook 等）
- 使用图标（icon）+ 文字的形式

### 6. 简繁切换
- 页面支持简体中文 / 繁体中文切换
- 默认繁体中文
- 所有文案需准备简繁两套文本

### 7. 明暗模式
- 页面支持亮色 / 暗色 / 跟随系统 三种模式切换
- 默认跟随系统（auto）
- 通过 CSS 变量或 Tailwind dark mode 实现

## Technical Notes

- **技术栈**：Astro + Tailwind CSS
- **页面标题**：蘇打綠 | Sodagreen
- **响应式设计**：必须适配所有设备尺寸（mobile-first）
- **性能优先**：背景图等大资源需优化加载
- **纯静态页面**：无需后端，可部署到任意静态托管服务
- **语言**：中文为主，默认繁体，可切简体

## Commands

- `npm run dev` — 启动开发服务器（端口 5353）
- `npm run build` — 构建生产版本
- `npm run preview` — 预览生产构建
